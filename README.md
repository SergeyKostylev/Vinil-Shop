<b>Проект:</b> интернет магазин написанный на Symfony 3.4 одни из основных возможностей представлены ниже:<br>
<b>Layout:</b> регистрация, добавление товара в корзину, редактирование корзины, оформление заказа...<br>
<b>AdminMenu:</b> работа с товарами, характеристиками, атрибутами, пользователями ...<br>
<b>Цель проекта</b> - приобретение практических навыков работы с Symfony Framework.
<br>
Сайт размещен по адресу http://vinil-music-shop.pp.ua .
